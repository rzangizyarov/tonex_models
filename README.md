# Репозиторий моделей TONEX

Репозиторий предназначен для хранения моделей усилителей, снятых с помощью **IK Multimedia TONEX**.

## Использование

Для использования моделей импортируйте соответствующие файлы пресетов (`.txp`) через **TONEX**.

Для обеспечения воспроизводимости поведения моделей использовать следующие уровни входного сигнала:

- **RMS**: -20.5 dB;
- **Peak**: -8.5 dB.

> Уровень замерен при следующих условиях:
>
> - **Peak Hold Time**: 1000 ms;
> - Игра медиатором средней-сильной атакой на риффе "Battery".

## Технологии

В репозитории используются следующие технологии и подходы:

- TONEX:
    - Используется для снятия и экспорта моделей усилителей и эффектов;
    - Все модели обучены в режиме Extended Training;
    - Обучение происходило на тензорных ядрах NVIDIA GeForce RTX 3050 .

- YAML:
    - Используется для хранения метаданных профилей.

- Pydantic:
    - Является источником истины для структуры метаданных;
    - Используется для валидации и обновления структуры метаданных.
