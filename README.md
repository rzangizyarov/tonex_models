# Репозиторий моделей TONEX

## Применение

Уровень входного сигнала для обеспечения воспроизводимости поведения моделей:

- **RMS**: -20.5 dB;
- **Peak**: -8.5 dB.

> Уровень замерен при следующих условиях:
>
> - **Peak Hold Time**: 1000 ms;
> - Игра медиатором средней-сильной атакой на рифе Battery.
